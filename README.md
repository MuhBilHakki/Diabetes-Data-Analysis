Metodologi yang digunakan 
Sumber data : Kaggle dataset penderita diabetes suku Indian Pima 1990
Tools Analisis : Microsoft Excel
Metode Analisis : Question -> Data Cleaning -> Exploratory Data Analysis -> Data Visualization -> Draw Conclusion
Variabel Kunci : Kehamilan, tekanan glukosa, tekanan darah, ketebalan kulit, insulin, umur, dan hasil audiens

Kesimpulan :
Berdasarkan hasil analisis bahwa wanita yang menderita penyakit diabetes bisa dipengaruhi oleh salah satu variabel kunci atau lebih
